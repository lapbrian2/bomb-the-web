# BOMB THE WEB — Urban Canvas

A graffiti wildstyle-inspired 3D landing page built with Three.js, GSAP, and pure CSS.

## Features

- **Toon-shaded 3D scene** — Cel-shaded blocks, spray cans, stars, and arrows with thick black outlines
- **Mouse parallax** — Scene responds to cursor movement with smooth lerp
- **Scroll zoom** — Mousewheel zooms the camera in/out
- **Click spray effect** — Click anywhere for a paint splatter burst
- **Custom cursor** — Animated dot + trailing ring with hover states
- **Loading screen** — Animated progress bar intro
- **Floating particles** — Colored paint dots drift upward
- **Responsive** — Works on mobile with fallback cursor
- **Zero dependencies** — Just Three.js and GSAP via CDN

## Run Locally

Just open `index.html` in a browser. No build step required.

## Tech Stack

- Three.js r128 (WebGL)
- GSAP 3.12 (animations)
- CSS custom properties
- Google Fonts (Black Han Sans, Oswald)
